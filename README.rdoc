= Tr8n Examples on Rails 3.0.x

= Installation Instructions

Run the following commands:

  $ git clone -b rails3 git@github.com:hatem/tr8n_examples.git
  $ cd tr8n_examples
  $ bundle
  $ rake db:migrate
  $ rake tr8n:init
  $ rails server

Now you can open your browser and point to:

  http://localhost:3000
